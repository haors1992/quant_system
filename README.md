# 量化交易系统

A股量化交易仿真系统 —— 基于规则驱动，机械执行，逃离人性弱点。

## 功能特性

- **多策略组合**：双均线、RSI、成交量突破，按权重聚合信号
- **全A股选股**：按成交额、涨跌幅、换手率、趋势评分筛选候选股
- **风控管理**：止损/止盈/移动止损/最大回撤控制，凯利公式简化版仓位管理
- **SQLite 持久化**：持仓、交易记录、资产快照跨日连续存储
- **多种运行模式**：实时交易 / 回测 / 选股筛选 / 定时运行
- **日报生成**：自动生成 Markdown 格式的每日交易报告

## 项目结构

## 快速开始

### 安装依赖

```bash
pip install akshare
```

### 运行

```bash
# 正常运行（交易日获取实时数据）
python main.py

# 全A股选股筛选（只看候选，不交易）
python main.py --screen

# 回测模式（用历史数据模拟）
python main.py --backtest

# 查看最新报告
python main.py --report

# 查看当前持仓和资产状态
python main.py --status

# 定时模式（每个交易日收盘后自动运行）
python main.py --schedule
```

## 策略说明

| 策略 | 权重 | 逻辑 |
|------|------|------|
| 双均线 | 0.4 | MA5 上穿 MA20 买入，下穿卖出 |
| RSI | 0.3 | RSI < 30 超卖买入，RSI > 70 超买卖出 |
| 成交量突破 | 0.3 | 成交量突破 20 日均量 1.5 倍 + 价格突破 |

信号强度 >= 0.5 时执行交易。

## 风控参数

| 参数 | 值 | 说明 |
|------|----|------|
| 单笔最大亏损 | 5% | 触发当日暂停交易 |
| 单日最大亏损 | 5% | 触发当日暂停交易 |
| 最大总回撤 | 20% | 触发后减半仓位 |
| 止损线 | -7% | 机械止损 |
| 止盈线 | +15% | 机械止盈 |
| 移动止损 | 5% | 从最高点回撤触发 |

## 数据源

- **实时行情**：新浪财经（批量获取，速度快）
- **K 线历史**：akshare（主源）/ 腾讯财经（备用）
- **交易日历**：内置 2025-2026 法定节假日
